# FlightCalculator

Work out how far you've flown


## Get running on your location machine

From the repo root, run the following commands:

```bash
cd src

python3 -m venv venv
. ./venv/bin/activate

python3 -m pip install --upgrade pip
pip install -r requirements.txt
```


## Using the app

From the repo root, run the following commands:

```bash
cd src
python3 main.py
```
